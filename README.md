# DROPLIE — Private Video Chat

## Features
✔ WebRTC camera + audio  
✔ Omegle style UI  
✔ Dark theme  
✔ Small local video + big remote video  
✔ Password required (gunapvt)  
✔ Messaging inside video room  

## Setup
npm install
npm start

Server runs on port 10000.
Open browser: http://localhost:10000
